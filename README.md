# Socializers - A Facebook-like Social Media Platform

Welcome to Socializers, a modern social media website inspired by Facebook. This platform allows users to connect with friends, share posts, photos, and stay updated with their social network.

## Features

### 🎯 Core Features
- **User Authentication**: Login and registration system
- **News Feed**: Scroll through posts from friends and pages
- **Post Creation**: Share text, photos, and videos
- **Stories**: Share temporary content that disappears after 24 hours
- **User Profiles**: Personal profiles with friends list and posts
- **Friend System**: Send and manage friend requests
- **Like & Comment**: Interact with posts through likes and comments
- **Share**: Share posts to your timeline or with friends
- **Search**: Find friends, posts, and content
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile

### 🎨 Design Features
- Modern, clean interface inspired by Facebook
- Smooth animations and transitions
- Dark/light mode support (planned)
- Accessibility features
- Mobile-responsive layout

## Technology Stack

### Frontend
- **HTML5**: Semantic markup for structure
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript (ES6+)**: Interactive functionality
- **Font Awesome**: Icon library
- **Responsive Design**: Mobile-first approach

### Backend (Planned)
- **Node.js**: Server-side runtime
- **Express.js**: Web framework
- **MongoDB**: Database for user data and posts
- **JWT**: Authentication tokens
- **Socket.io**: Real-time features

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Installation
1. Clone or download the project files
2. Navigate to the project directory
3. Open `index.html` in your web browser
4. Start exploring Socializers!

### Development Setup
For a better development experience, you can use a local web server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server

# Using PHP
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser.

## Usage Guide

### Navigation
- **Home**: Main news feed with posts from friends
- **Profile**: Your personal profile page
- **Friends**: Manage your friends list
- **Photos**: View and upload photos
- **Videos**: Watch and upload videos
- **Events**: Discover and create events
- **Games**: Play social games

### Creating Posts
1. Click on "What's on your mind?" in the post creation area
2. Write your post content
3. Add photos, videos, or feelings (optional)
4. Click "Post" to share

### Interacting with Posts
- **Like**: Click the Like button to show appreciation
- **Comment**: Click Comment to join the conversation
- **Share**: Click Share to spread the content

### Managing Friends
- Browse friend suggestions in the right sidebar
- Click "Add Friend" to send friend requests
- Accept or decline incoming requests

## Keyboard Shortcuts

- **Ctrl/Cmd + K**: Focus on search bar
- **Escape**: Clear search and unfocus
